# RemoteSystemMonitor Android App (Lightweight)

This is a lightweight Android WebView app that displays your existing RemoteSystemMonitor dashboard.

## Open in Android Studio
1. Open Android Studio
2. `File` -> `Open`
3. Select: `android-app`
4. Let Gradle sync finish
5. Run on phone/emulator

## Default URL
- `http://192.168.29.180:8080`
- You can change it in the top URL box inside the app.
- The app remembers your last URL.

## Notes
- Your phone must be on the same WiFi as your PC server.
- Keep `monitor.bat` server running on your PC.
